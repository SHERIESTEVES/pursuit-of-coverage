PURSUIT OF COVERAGE WEBSITE

1. Open index.html to preview the site.
2. Upload all files to the root of a GitHub Pages repository or web host.
3. Update the email address and phone number in contact.html.
4. Replace the remote Unsplash images with licensed company images if desired.
5. Connect the contact form to Formspree, HubSpot, Jotform, or another form provider for direct submissions.
